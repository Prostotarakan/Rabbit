For matlab version 9.4 (R2018a) or higher

English part:
The implemented application generates a map, places hares and wolves on it.
If hares are born without dividing by sex, then wolves have a division and new wolves appear only when a female and a male meet.
And so, everyone needs to eat in order to survive, everyone cheerfully run in different directions.


Русская часть:
Реализованное приложение генерирует карту, расставляет по ней зайцев и волков. 
Если зайцы зарождаются без деления на пол, то у волков есть разделение и новые волки появляются только при встрече самки и самца. 
А так, всем необходимо питаться, чтобы выжить, все бодро бегают в разные стороны.
